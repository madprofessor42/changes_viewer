# Changes Viewer

VS Code расширение для локальной истории изменений файлов.

## Описание

Changes Viewer автоматически отслеживает изменения файлов в редакторе VS Code и от внешних процессов, создает снапшоты и предоставляет удобный интерфейс для просмотра, сравнения и восстановления истории изменений.

## Возможности

- Автоматическое отслеживание изменений файлов (typing, save, filesystem)
- Просмотр истории изменений через Timeline API
- Восстановление файлов к предыдущим версиям
- Сравнение версий файлов
- Управление историей (принятие изменений, очистка)

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Скомпилируйте проект: `npm run compile`
4. Запустите в режиме разработки: нажмите F5 в VS Code

## Разработка

### Структура проекта

```
src/
  ├── extension.ts          # Точка входа
  ├── types/               # TypeScript типы
  ├── services/            # Сервисы (Storage, Cleanup, etc.)
  ├── providers/           # Timeline Provider
  ├── commands/            # Команды расширения
  ├── utils/               # Утилиты
  └── migrations/          # Миграции данных
```

### Команды

- `npm run compile` - компиляция TypeScript
- `npm run watch` - компиляция в режиме наблюдения
